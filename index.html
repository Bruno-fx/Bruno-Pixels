Case Study: Kigali FreshCart

The Problem: Kigali FreshCart suffered from high perishable waste and frequent stockouts due to manual inventory tracking.

The Solution: We built an end-to-end forecasting pipeline. By integrating machine learning models with supplier APIs, we created an automated reorder engine that learned from local demand patterns.

Key Results:
  • Accurate short-term demand forecasts
  • Fully automated ordering workflows
  • Live operational KPI dashboard
  • 18% reduction in perishable waste within 3 months
